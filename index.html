How to apply

  1. Click Download CSS and save the generated file.
  2. Put the CSS file under /config/www/.
  3. Open Home Assistant, then go to Settings -> Dashboards -> menu -> Resources.
  4. Add /local/homeassistant-custom-font.css as a Stylesheet resource.
  5. Restart Home Assistant, then hard refresh the browser or clear the frontend cache.
